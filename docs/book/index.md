## What You'll Build

By the end of this book, you'll have a fully functional blog with:

- Public blog posts with markdown support
- Admin authentication
- CRUD operations for posts
- Categories and tags
- JSON API endpoints
- Static file serving

## Prerequisites

- Basic TypeScript knowledge
- Bun installed (v1.0.0+)
- A text editor
- Terminal access

## Source Code

Each chapter includes complete, runnable code examples. The final blog application source is available in the `examples/blog` directory.
